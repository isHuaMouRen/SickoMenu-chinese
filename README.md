## 汉化版安装

> [!caution]
> **汉化版仅支持 `Steam` 和 `Itch.io` 版本！(x86)**

在安装汉化版之前，请阅读[官方储存库](https://github.com/g0aty/SickoMenu)的自述文件，确保官方版可以正常安装并运行

确保官方版运行无误后，前往当前仓库的[Releases](https://github.com/isHuaMouRen/SickoMenu-chinese/releases)页面，检查最新版版本号是否与当前安装的官方版版本号相同，如不相同可能会引发一些问题

所有注意事项无误后，直接将汉化版文件覆盖至游戏根目录
